vw and vh unit

vw and vh stands for viewport width and viewport height.
it's useful for making element size exactly as the viewport size

example:

.fullScreen {
	width: 100vw;
	height: 100vh;
}
.quarter {
	font-size: 25vh;
	line-height: 25vh;
}
.halfWindowDiv {
	width: 50vw;
	height: 100vh;
}

transform and transition

transform is useful for animating elements
it is preferable to use this for moving animated element than using position property

example:

.moveRight1s {
	transform: translateX(0);
	transition: 1s;
}
.moveRight1s:hover {
	transform: translateX(100px);
}
.zoom2x {
	transform: scale(1);
	transition: 1s
}
.zoom2x:hover {
	transform: scale(2);
}
.zoomAndRotate360deg {
	transform: rotate(0deg) scale(1);
	transition: 1s
}
.zoomAndRotate360deg:hover {
	transform: rotate(360deg) scale(2);
}

em

em is useful for responsive design
it scales proportionally from the parent's font-size (very useful when we were using media queries)
it also can be use for the usual size unit for width, height, margin, padding, etc

example:

.grandParent {
	font-size: 16px
}
.grandParent .parent {
	font-size: 2em;
}
.grandParent .parent .child {
	font-size: 0.5em;
}

I am the child 

.1emMargin {
	margin-top: 1em;
}
.2emWidth {
	width: 2em;
}

calc

calc is useful to perform calculation in css
we can use it to combine css units or anything that needs mathematical method

example:

.100pxFromCenter {
	margin-top: calc(50vh + 100px)
}

.exactlyThird {
	width: calc(100% / 3)
}

.doubleTheCurrentFontSize {
	font-size: calc(1em * 2);
}
